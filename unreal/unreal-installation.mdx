## Install LCK in your project

<Accordion title="0.9.1 Compatibility" defaultOpen icon="ballot-check">
  **Game Engine**

  - Unreal Engine 5.4.4+

  **Platforms**

  - Meta Quest 2/3/3s
  - Windows PCVR

  **Graphics API**

  - Vulkan, DX11, OpenGL

  **Audio**

  - Vivox SDK version 5.26.0.unr.0
</Accordion>

### Download LCK and get your Tracking ID

1. Go to the [**LIV Developer Dashboard**](https://dashboard.liv.tv/dev/login?welcome=false).
2. Set up your organisation and game on the portal.
3. Download the SDK and save the Tracking ID provided.

### Install LCK

1. Unzip the LCKGame project

   <img
     src="/images/unreal-images/image27.png"
     alt="Image27"
     title="Image27"
     style={{ width:"29%" }}
   />
2. Generate Visual Studio project files

   <img
     src="/images/unreal-images/image19.png"
     alt="Image19"
     title="Image19"
     style={{ width:"67%" }}
   />
3. Open the .sln generated and build the project.

   <img
     src="/images/unreal-images/unreal-image1.png"
     alt="Unreal Image1"
     title="Unreal Image1"
     style={{ width:"67%" }}
   />
4. You can find the LCK plugins inside the /Plugins folder.
   - LCK
   - LCKOboe
   - LCKTablet
   - LCKUI
   - LCKUnrealAudio
   - LCKVivox
   - LCKWwise
5. Optional Plug ins

If you're using Vivox or Wwise, you'll need these optional plug ins

![Optional1](/images/unreal-images/optional1.png)
