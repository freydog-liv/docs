## Description

`LckQualityConfig` is a Unity `ScriptableObject` that holds configuration data for quality settings across different platforms (Android, Desktop). It provides a structured way to define base options, device-specific overrides, and retrieve the appropriate quality options at runtime depending on the system.

## Usage

Assign `LckQualityConfig` in your project to manage capture quality options across devices.\
When calling `GetQualityOptionsForSystem()`, the config automatically detects the platform and, if applicable, the current device model to return the correct set of quality options.

## Examples

```
var config = Resources.Load<LckQualityConfig>("LckQualityConfig");
var qualityOptions = config.GetQualityOptionsForSystem();

foreach (var option in qualityOptions)
{
    Debug.Log($"Option: {option.Name} (Default: {option.IsDefault})");
}
```
