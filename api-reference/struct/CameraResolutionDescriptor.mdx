## Description

[`CameraResolutionDescriptor`](/api-reference/struct/CameraResolutionDescriptor) specifies the width and height of a camera’s output resolution.
It is used as part of a [`CameraTrackDescriptor`](/api-reference/struct/CameraTrackDescriptor) to define how video frames should be captured and encoded.

By default, it initializes to a **square resolution of 512 × 512 pixels**.
