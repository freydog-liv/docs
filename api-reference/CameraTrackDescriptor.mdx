## Description

**CameraTrackDescriptor** is a struct that encapsulates key parameters required for configuring a camera track in LCK. It holds a [CameraResolutionDescriptor](/api-reference/cameraresolutiondescriptor) to define resolution settings and also includes bitrate and framerate parameters for both video and audio. Default values are provided for video bitrate, frame rate, and audio bitrate to simplify common scenarios.

## Usage

When creating a **CameraTrackDescriptor**, you typically need to supply a [CameraResolutionDescriptor](/api-reference/cameraresolutiondescriptor). Unless you need custom settings, you can rely on the default values for video bitrate (`5 << 20`), framerate (`30`), and audio bitrate (`192000`).

## Examples

```
// Using default bitrate, framerate, and audio bitrate
var resolution = new CameraResolutionDescriptor(/* ... resolution params ... */);
var descriptor = new CameraTrackDescriptor(resolution);

// Custom configuration
var customDescriptor = new CameraTrackDescriptor(
    resolution,
    bitrate: 10 << 20,     // 10 Mbps
    framerate: 60,          // 60 fps
    audioBitrate: 256000    // 256 kbps
);
```
