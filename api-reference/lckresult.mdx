---
title: "LckResult<T> Class"
description: "Data container returned by LckService."
---

Data container returned by `LckService` methods, can be queried for information and success/failure status.

### Properties

| Name      | Type        | Description                                                                      |
| :-------- | :---------- | :------------------------------------------------------------------------------- |
| `Success` | `bool`      | Indicates whether the operation was successful.                                  |
| `Message` | `string`    | Provides an optional message, typically used for error information.              |
| `Error`   | `LckError?` | Contains the error code or type if the operation failed.                         |
| `Result`  | `T`         | The result value of the operation if successful; default value of `T` otherwise. |

### Static Methods

#### `LckResult NewSuccess()`

Creates a successful result with no returned value.

```c#
var result = LckResult.NewSuccess();
```

- **Returns**: An instance of `LckResult` indicating success.

---

#### `LckResult NewError(LckError error, string message)`

Creates a failed result with an error and a message.

```c#
var result = LckResult.NewError(LckError.SomeError, "An error occurred.");
```

- **Parameters**:
  - `error`: An `LckError` enum or struct representing the error type.
  - `message`: A descriptive error message.
- **Returns**: An instance of `LckResult` indicating failure.
