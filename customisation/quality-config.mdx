The LCK tablet allows you to customise the quality settings in a variety of ways through the [LckQualityConfig](/customisation/quality-config) scriptable object, stored in the Quality Config field in the **LCKCameraController** script.

## Lck Quality Config

This scriptable object provides a way to define different quality presets for Android and Desktop platforms and define device specific overrides.

### Quality Options

For each platform you can store different quality presets that can be used by the tablet. By default this is set to the SD/HD for android and SD/HD/2k/4k for windows.

Each quality preset has a name field and a [CameraTrackDescriptor](/api-reference/struct/CameraTrackDescriptor)  that allows you to set the Bitrate, Framerate, Audio Bitrate and Width and Height through a [CameraResolutionDescriptor](/api-reference/struct/CameraResolutionDescriptor) .
